# Python课程管理系统（源码+实验报告）

## 核心功能
- 登录注册
- 学生选课
- 学生退课
- 课程详情浏览
- 教师发布课程
- 查看课程学生
- 课程信息管理
- 课程用户信息总览

## 适用场景
- 教育机构
- 课程学习平台
- 教学管理

## 技术特点
- 基于 Flask 框架
- 使用 MySQL 数据库
- 前端采用 HTML、CSS 和 JavaScript 编写
- 代码可运行，需自行配置环境
